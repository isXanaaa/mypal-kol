# Mypal Creator Partnership Hub

This repository contains a static landing page for Mypal BD creator and KOL outreach.

## Files to publish

- `index.html` is the public entry page.
- `assets/` contains all images used by the page.
- `.nojekyll` disables Jekyll processing on GitHub Pages.
- `kol-partnership.html` is a local backup/work file and is not required for publishing.

## GitHub Pages setup

1. Create a public GitHub repository, for example `mypal-kol`.
2. Upload `index.html`, `.nojekyll`, `README.md`, and the full `assets/` folder.
3. Open the repository `Settings`.
4. Go to `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. Wait a few minutes, then open the published Pages URL.

The public URL should look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/mypal-kol/
```
